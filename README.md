# Koliseo Issue Tracker

Welcome to the public issue tracker for [Koliseo](https://www.koliseo.com), an event management platform.

## About Koliseo

Koliseo is a comprehensive event management platform that helps organizers manage conferences, meetups, and other events. It provides features for ticket sales, attendee management, speaker coordination, and more.

## Purpose of This Repository

This repository is dedicated to tracking issues, bugs, and feature requests from Koliseo users who do not have a support contract. If you're experiencing problems with the platform or have suggestions for improvements, this is the place to report them.

## How to Submit an Issue

1. **Search existing issues** - Before creating a new issue, please check if a similar issue already exists
2. **Create a new issue** - Click on the "Issues" tab above, then click "New Issue"
3. **Provide detailed information** - Include:
   - A clear and descriptive title
   - Steps to reproduce the problem (for bugs)
   - Expected behavior
   - Actual behavior
   - Screenshots or error messages if applicable
   - Browser/device information if relevant

### Issue Types

We use the following issue types:

- **bug** - Something isn't working as expected
- **feature-request** - Suggestion for new functionality
- **documentation** - Related to missing or incorrect documentation
- **question** - General questions about the platform

## Support

- **Users with support contracts**: Please use your designated support channels
- **Community support**: Community support is managed through this repository
- **Security issues**: Please report security vulnerabilities privately to security@koliseo.com

## Code of Conduct

Please be respectful and constructive in all interactions. We're here to help make Koliseo better for everyone.
